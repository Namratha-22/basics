The sorted() function returns a sorted list of the specified iterable object.

You can specify ascending or descending order. Strings are sorted alphabetically, and numbers are sorted numerically.

Note: You cannot sort a list that contains BOTH string values AND numeric values

This function will not affect the interable in the variable . It will just dispalys

The iterable can be list, tuple,string, set but the output is always a list.

Syntax: sorted(iterable, key=lambda/def, reverse=Default false)

Ex:
a={1,5,9,6,3,4,8,2}
print(sorted(a))

Sol:
[1,2,3,4,5,6,8,9]

Ex:
a=(1,5,9,6,'a',4,8,'d')
print(sorted(a))

Ex:
['1', '2', 'd', 'f', 'g', 'j']
